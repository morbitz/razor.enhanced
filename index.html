Welcome to Razor Enhanced!

This site houses the Razor Enhanced project and documentation. Currently the project is in its beta phase and is considered very stable. It is actively being worked on and improved.

Differences between Razor Enhance and the original Razor:

  • It is based on .NET 4.8 and has therefore a lot of improvements available in this framework (higher speed, stability, etc.)
  • A wide range of specialized Agent (Autoloot, Scavenger, Organizer, Vendor Buy, Sell Vendor, Dress, Restock, Bandage, etc.)
  • A new targeting system and filtering evolved that allows you to select more accurately the targets in PVP and PVM
  • A built-in scripting engine based entirely on IronPython .NET that lets you use all the native features of the Python language with ability to run multiple scripts simultaneously
  • A built-in scripting engine that mimics UOSteam command and syntax.
